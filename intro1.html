Introduction To Computer


Computer is an advanced electronic device that takes data and intruction and gives meaningful reasult to user.It store large amount of data permanently.It perform arithmatic and logical calculation.

Functions of Computer

Computer

Input

The data and instruction what we fed into the computer is called Input.

Processing

Treating the data according to the instruction of user is called Proccesing.

Output

The processed data given by the computer after processing is called Output.

Storage

Computer has storage place that stores large amount of data and information permanetly.HDD,CD,DVD,BD are used to store the information in the computer.
